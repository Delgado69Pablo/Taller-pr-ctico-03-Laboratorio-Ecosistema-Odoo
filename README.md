# Taller práctico 03: Laboratorio: Ecosistema Odoo

### Descripción

En esta práctica se ha trabajado con el ERP Odoo para aprender a instalar módulos, importar datos, generar informes y exportar información.
El objetivo principal es entender cómo se conectan los diferentes módulos de Odoo y cómo el sistema gestiona automáticamente la información utilizando PostgreSQL.

## Objetivos

- Instalar y configurar módulos en Odoo.
- Conectar los módulos de Ventas, Inventario y Facturación.
- Importar clientes desde un archivo CSV.
- Modificar informes PDF usando QWeb/XML.
- Exportar información para utilizarla en otros sistemas.
- Comprender cómo Odoo organiza y relaciona los datos.

## Tecnologías utilizadas

- Odoo
- PostgreSQL
- Docker
- Docker Compose
- Visual Studio Code
- QWeb / XML
- CSV
